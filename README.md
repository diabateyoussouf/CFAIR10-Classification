# CFAIR10-Classification TP accadémique
Le problème consiste à résoudre une classification multiclasse sur 10 classes (par exemple : dog, cat, airplane, etc.). Nous avons utilisé K-Fold pour identifier le meilleur modèle.  Nous avons d’abord testé un modèle séquentiel basique (couches Dense après Flatten), mais il n’a pas permis d’obtenir de bonnes performances. Même en utilisant un modèle pré-entraîné comme ResNet50 sur des images 32×32, les résultats restaient limités.  La solution efficace a été d’adopter un CNN composé de 4 blocs convolutifs, capable d’extraire les caractéristiques locales des images et d’améliorer significativement la performance du modèle.
